# Opinion System (舆情监测系统)

## 系统概述
**OpinionSystem** 是一套全链路舆情分析一站式平台。从数据采集、智能筛选到多维分析与解读，以全功能体验帮助团队快速洞察舆情脉络，沉淀可复用的决策资产。

## 核心关键词
- **THROUGHPUT**: 舆论流体力学，传播趋势洞察。
- **RETRIEVAL**: GraphRAG + Vector 双引擎智能检索。
- **PROJECTS**: 10+ 专题，支持多专题并行调度。

## 硬核能力
一体化的舆情工作台，覆盖采集、处理、分析、检索与报告输出，每个环节都保持极简界面与可靠的工程能力。

1.  **自动化数据流水线**: 采集、清洗、入库全流程自动化，保障数据质量与时效性。
2.  **AI 智能筛选**: 由 Qwen 驱动的相关度筛选与内容分类，高效过滤噪声信息。
3.  **多维度分析**: 情感、地域、趋势、关键词等多维指标一站式呈现。
4.  **可视化洞察输出**: 自动生成高层可读的图表与摘要报告，支持导出复用。
5.  **智能解读引擎**: TagRAG 背景知识结合大模型，为舆情团队提供深度观点。
6.  **多形态智能检索**: GraphRAG 与向量检索协同，精准定位关键内容与证据。

## 内容分析 · 精准到字段
内容编码分析引擎以 **Qwen-Plus** 为核心，支持动态字段建模，自动适配不同专题的数据结构。
- **功能亮点**: 动态字段适配、高准确度内容编码、50 QPS 批量处理吞吐、Excel/JSON 双形态输出。
- **处理流程**: 多源接入 -> 结构化清洗 -> 智能编码 -> 指标渲染。

## 智能解读 · AI 级洞察官
将 Qwen-Plus 与 **TagRAG** 知识库深度融合，为每个专题输出可信赖的专业分析报告。
- **系统特性**: AI 深度解读、TagRAG 背景召回、高并发处理 (50 并发)、全链路追踪。
- **解读维度**: 
    - 情感分析 (Attitude)
    - 话题分类 (Classification)
    - 地域分析 (Geography)
    - 关键词分析 (Keywords)
    - 发布者分析 (Publishers)
    - 趋势分析 (Trends)
    - 声量分析 (Volume)
    - 内容分析 (ContentAnalyze)

## 示例：控烟专题
- **信息类别**: 控烟立场 / 烟草立场 / 其他
- **议题编码**: 1-1 烟草与健康、1-2 无烟立法等
- **信源编码**: 卫生部门、媒体自采、意见领袖等
- **报道体裁**: 事实性报道、观点性报道、深度综合
- **诉求方式**: 恐怖诉求、人性诉求、行动呼吁等
