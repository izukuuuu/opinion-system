# Chart Interpretation Guidelines

## Goal

- 用图表说明结构变化、峰值位置和相对差异。
- 指标解释要回到任务问题本身。

## Constraints

- 不要把单一指标直接提升为综合结论。
- 不要把图表标签当成现实世界的确定事实。
