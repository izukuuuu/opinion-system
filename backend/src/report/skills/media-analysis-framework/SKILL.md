# Media Analysis Framework

## Goal

- 在传播路径分析中区分官方媒体主导信源与地方媒体的发声结构。
- 在立场冲突分析中识别哪些声音由具备更高信度的媒体发出。
- 量化官方媒体发文占比，评估事件的"官方引导力度"。

## Tool Hints

`media_coverage_summary_tool` — 调用时机与使用规范：

**何时调用：**
- 需要评估"官方引导力度"时（官方媒体发文占比高 → 引导明确）。
- 需要识别传播节点的来源信度时（官方媒体转发 → 放大效应更确定）。
- 需要判断立场是否由国家级媒体主导还是地方声音发散时。
- 任何涉及"媒体生态"、"媒体集中度"、"来源结构"的分析场景。

**参数说明：**
- `topic_identifier`：与 `normalized_task.json` 中的 `topic_identifier` 字段保持一致。
- `start_date` / `end_date`：对应本次分析的时间边界，格式 `YYYY-MM-DD`。

**结果解读：**
- 官方媒体占比 > 60%：官方引导力度显著，信息生产高度集中。
- 官方媒体占比 30%-60%：双轨并行，官方声音存在但未主导。
- 官方媒体占比 < 30%：以地方/民间声音为主，事件可能处于官方关注盲区。

**当工具返回"未找到媒体打标结果"时：**
- 不要将媒体覆盖分析作为必要前提阻塞报告生成。
- 在传播结构结论中注明"因媒体打标数据尚未完成，来源信度维度暂缺"。
- 继续基于证据卡和其他指标进行分析。

## Constraints

- 禁止将官方媒体数量直接等同于"信息可信度高"——来源结构只是维度之一。
- 官方媒体发文量大不等于民间认同，需结合情感和立场数据交叉判断。
- 工具返回的"Top 10 媒体列表"仅用于结构说明，不要在报告正文中全量列举。
- 不要以"地方媒体"标签否定内容质量，只描述来源多样性和集中度事实。
