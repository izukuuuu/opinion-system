# 正式报告事实写法约束

该 skill 用于统一约束正式报告正文的事实优先写法，适用于政策动态、公共热点、危机响应、专题分析等多类报告。核心目标不是“优化措辞”，而是阻止三类内容混入正式稿：

- 编辑批注语
- 写作指导语
- 无证据支撑的宏观阶段判断

## Goal

将事件、政策、回应和争议按时间顺序组织为正式报告文本，准确呈现时间、主体、文本内容和争议焦点，不加入作者批注、写作说明和越级宏观判断。

## Hard Constraints

1. 正文只写事件事实、规则内容、执行安排和明确争议，不写”这段应该如何写”。
2. 禁止使用”不宜写成””应当表述为””更适合写成””不能只写成””不应处理为”这类批注句式。
3. 禁止使用”起点””转折点””底噪””制度性触发点””议题位置””传播意义””政策想象””比较框架”这类抽象概括词，除非后文立即给出可核验对象。
4. 禁止把若干事件直接上推为”全年阶段变化””整体认知结构变化”或”舆论已进入某阶段”，除非材料中已经存在明确的时间序列证据。
5. 每段至少覆盖以下四项中的三项：时间、主体、文本、争议焦点。
6. 争议焦点必须写成可直接复述的问题，不得只写空泛概念。
7. 解释句只允许覆盖本段已出现的事实，不得越级推导到整轮舆情结构、公众认知转向或长期阶段变化。
8. 用词优先选择”列入、规定、回应、实施、审议、通过、刊宪、纳入、禁止、要求、设置、处罚”等可核验动词，避免”推进、强化、外扩、重塑、显化、转化”等高抽象动词直接充当主判断。
9. 图表段落必须服从”先判断、后证据”的总分结构，不得以”这张图不是……””图X并不是……”这类辩护句开头。
10. 禁止替图表辩护：不得写”这张图不是简单的……””图表的真正含义是……””不要只看这张图的……”。
11. 禁止替文字辩护：不得写”文字的作用就在于……””这段话不只是……而是……””真正要说明的是……”。
12. 禁止替读者辩护：不得写”读者可能会误以为……””不能只理解成……””看这张图时要注意……”。
13. 禁止解释动词跳级：不得用”说明””表明””显示出”后面接宏观判断（如”说明公众开始……””表明治理已转入……”）。

## Banned Patterns

以下句式默认视为不进入正式正文：

### 批注语（讨论”如何写”）

- 不宜处理为……
- 应当表述为……
- 更适合写成……
- 不应写成……
- 不能只写成……
- 这段话的作用在于……
- 文字的作用就在于……
- 写这段是为了……

这些句子是编辑批注，讨论的是写作方法，不是事实本身。

### 辩护语（替图表、替文字、替读者辩护）

**替图表辩护：**

- 这张图并不是……
- 这张图不是简单的……
- 图X并不是一个简单的……
- 这张图真正说明的不是……而是……
- 图表的真正含义是……
- 不要只看这张图的……
- 这张图不只是……更是……

**替文字辩护：**

- 文字的作用就在于……
- 这段话不只是……而是……
- 写到这里不是为了……而是为了……
- 真正要说明的是……
- 不能只理解成……
- 这句话的真正意图是……

**替读者辩护：**

- 读者可能会误以为……其实……
- 不要把这张图理解成……
- 不能只看成……
- 看这张图时要注意……
- 这张图容易被误解为……

这些句子都是在”预先反驳假想的误读”，不是陈述事实。正式报告不负责替读者纠正理解，只负责把事实写清楚。

### 宏观跳级语（越过证据直接下判断）

- 这一节点的作用在于……
- 其重要性不只是……而是……
- 真正推动讨论扩散的是……
- 这标志着议题进入新阶段……
- 这重塑了议题的认知结构……
- 这使地方差异被转化为全国讨论……
- 起始底噪
- 更强硬的外部参照
- 全国舆情素材
- 第一波政策触发点

这些句子直接跳到宏观解释层，不是对事实本身的稳定表述。

### 解释动词泛滥

- 说明……（除非后面接具体事实）
- 表明……（除非后面接具体事实）
- 显示出……（除非后面接具体事实）
- 意味着……（禁止用于跳级判断）
- 意味着议题进入……（禁止）
- 说明公众开始……（禁止）
- 表明治理已转入……（禁止）

正确用法：该回应显示目前不存在全国统一规定。（”显示”后面是具体事实）
禁止用法：这表明公众开始重新理解空间边界。（”表明”后面是跳级判断）

## Required Content Units

正式报告正文优先保留四类信息单位：

1. 时间
   写明月份、日期、实施时间、回应时间、审议时间、通过时间、刊宪时间。
2. 主体
   写明谁发布、谁回应、谁修法、谁执法、谁发生冲突、谁提出争议。
3. 文本
   写明条例、草案、规范、通知、回应中具体新增、明确、调整或要求了什么。
4. 争议焦点
   写成可直接复述的问题，例如“某类场景是否纳入禁烟范围”“某类经营者是否负有劝阻义务”“某系统是否存在统一规定”。

## Allowed Explanation Scope

可以写解释，但解释只能停留在文本直接支持的层级。

允许：

- 该草案将排队等候进入指定场所纳入禁烟安排。
- 该标准将某类室外区域纳入无烟场所建设范围。
- 该回应显示目前不存在全国统一禁烟规定。

不允许：

- 这标志着治理进入新阶段。
- 这使地方差异被正式转化为全国讨论。
- 这说明公众开始重新理解空间边界。
- 这重塑了议题的认知结构。

前一组是事实和最低限度归纳，后一组已经越级进入宏观解释。

## Sentence Rules

### 1. 主句必须落在事实上

优先写成：

“某年某月某日，某主体发布了某文件，并明确了某项要求。”

不要写成：

“某主体在这一节点提供了制度参照。”

### 2. 抽象判断后面必须接具体对象，否则删除

例如写“规则争议”时，后面必须能明确交代：

- 哪条规则
- 哪类场景
- 哪个主体
- 哪个争点

如果接不出来，就不要保留这个抽象词。

### 3. 一句只承担一层功能

不要在同一句里同时完成三件事：

- 叙述事件
- 解释意义
- 概括整轮变化

正文宜保持“先事实，后有限归纳”，且归纳只覆盖本段材料，不跨越到整轮结构。

### 4. 图表段落不得辩护或自我解释

图表段落的核心问题是：写作者习惯性地替图表”辩护”，担心读者误读，于是先说”这张图不是什么”，再说”这张图真正是什么”。这种写法是自我解释，不是事实陈述。

**禁止的辩护结构：**

- 先否定一个假想的误读（”这张图不是简单的……”）
- 再给出”真正”的含义（”而是……”）
- 最后解释文字的作用（”文字的作用就在于……”）

这种结构把图表当成需要辩护的对象，把正文当成需要自我解释的文字。正式报告不这样做。

**正确的图表写法结构：**

1. 先给出本段判断（”按时间趋势，2月至5月……”）
2. 再指出图中对应的具体数据（”8月25日当周总量达3742”）
3. 如有必要，补充最小归纳（仅覆盖本段材料）

**错误示例：**

> 图3并不是一个简单的总量折线，而是几个主题簇在不同时间段轮流主导讨论的结果。按当前时间趋势看，2月至5月持续时间最长的主线是”区域控烟政策与法规”，说明上半年讨论更多围绕地方规则、修法和制度推进展开。因此，这张图真正说明的不是”热度反复波动”这么简单，而是控烟议题在年内多次发生讨论对象切换。文字的作用就在于把这些主题簇分别解释清楚。

**正确示例：**

> 按时间趋势，2月至5月持续时间最长的主线是”区域控烟政策与法规”，上半年讨论围绕地方规则、修法和制度推进展开；8月18日当周，”特定场所禁烟管理”显著抬升，高铁站台等具体空间成为讨论焦点；8月25日当周出现全年最强波峰，总量达3742，其中”社会热点事件与舆论”一项为3022；11月至12月，除制度推进线外，”戒烟方法与辅助产品”等主题也开始出现。

对比：

- 错误版：先辩护（”不是简单的……”），中间夹”说明”，结尾再辩护（”真正说明的不是……文字的作用就在于……”）
- 正确版：直接给出判断和数据，删除所有辩护句和”说明”类解释动词

**判断是否违规的快速方法：**

删掉所有”不是、而是、真正、作用、说明、表明”这类词，看正文是否仍然成立。如果删掉后信息完整，说明这些词本来就是空转的辩护话语。

## Output Style

- 使用正式报告语体。
- 不写作者立场，不写写作说明，不写编辑批注。
- 每段围绕一个时间节点或一个事实簇展开。
- 先写事实，再写对应的具体争议问题。
- 允许有限归纳，但归纳对象仅限本段材料，不上升为整体周期判断。

## Quick Check

一段话写完后，做三项检查：

### 1. 删除空转词

删掉所有”作用、意义、阶段、位置、转折、起点、应当、不宜”这类词，正文是否仍然成立。

如果删掉之后信息几乎不受影响，说明这部分本来就是空转的写作话语，不应进入正式稿。

### 2. 删除辩护句

删掉所有”不是……而是、真正说明的、文字的作用、图表的含义”这类辩护句，正文是否仍然成立。

如果删掉后数据和判断完整保留，说明辩护句纯属多余。

### 3. 删除解释动词跳级

删掉所有”说明、表明、显示出、意味着”后面接的宏观判断，看是否只剩下具体事实。

正确：该回应显示目前不存在全国统一规定。（删掉后仍有具体事实）
错误：这表明公众开始重新理解空间边界。（删掉后只剩空判断）
